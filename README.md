# React Currency Converter 💱

A modern, fast, and responsive currency converter web application built with React, inspired by platforms like XE.com. 

## 🚀 Tech Stack

* **Frontend Framework:** [React](https://reactjs.org/)
* **Build Tool:** [Vite](https://vitejs.dev/) (for fast Hot Module Replacement and optimized builds)
* **Styling:** [Tailwind CSS](https://tailwindcss.com/) & PostCSS
* **Language:** JavaScript

## ✨ Features

* **Real-time Conversion:** Seamlessly convert between different currencies.
* **Modern UI:** Clean, intuitive, and highly responsive interface designed for both desktop and mobile.
* **Fast Performance:** Powered by Vite for lightning-fast load times and a smooth developer experience.
